# Sistema-Bancario-Python
Sistema bancário via terminal, feito em python com login, depósito, saque e histórico

Este projeto foi desenvolvido com o intuito de registrar minha evolução e aprendizado.
O exercício proposto desafia a utilização de módulos e funções, dando início as boas práticas da programação

ESTRUTURA:
Separei o sistema em duas etapas
banco_funcoes.py    -  Onde estão as funções utilizadas no sistema.
sistema.py          -  Ponto principal do sistema

APRENDIZADO:
Foi um exercício interessante de ser desenvolvido, estava praticando exercícios para lógica de programação e decidi me desafiar:
-  Organização da estrutura do código, em funções e módulos
-  Parâmetros entre funções
-  Boas práticas iniciais
  
Estou em busca de me desenvolver na programação e seguirei aprendendo, muito obrigado por visitar!
